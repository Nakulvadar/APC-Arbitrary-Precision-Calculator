# APC-Arbitrary-Precision-Calculator

## Project Overview

A C-based Arbitrary Precision Calculator that performs addition, subtraction, multiplication, and division on large integers using doubly linked lists. This project is designed to overcome the limitations of built-in data types by handling extremely large numbers using Data Structures.

---

## Features

- Add large integers  
- Subtract large integers  
- Multiply large integers  
- Divide large integers  
- Handles very large numbers beyond `long long`  
- Implemented using Doubly Linked Lists  
- Modular and structured programming  
- Dynamic memory allocation  
- Command-line based execution  
- Accurate sign handling  

---

## Technologies Used

- Programming Language: C  
- Data Structure: Doubly Linked List  
- Compiler: GCC  
- Platform: Linux / Windows  

---

## How to Run

1. **Clone the repository:**
```bash
git clone <your-repository-url>
```

2. **Navigate to the project folder:**
```bash
cd APC-Arbitrary-Precision-Calculator
```

3. **Compile the project:**
```bash
gcc *.c -o apc
```

4. **Run the program:**
```bash
./apc
```
